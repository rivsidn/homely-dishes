## 原料

| 原料         | 用量                 |
| ------------ | -------------------- |
| 豆腐         |                      |
| 葱           | 多一些，后边分两次下 |
|              |                      |
| 五花肉[可选] |                      |
|              |                      |
| 八角         | 一瓣                 |
|              |                      |
| 酱油         |                      |
| 油           | 多一些，煎豆腐不糊锅 |
| 盐           |                      |
| 淀粉         |                      |



## 制法

* 加入油
* 加入八角、五花肉，煸五花肉至出油
* 加入一半葱，煸黄、煸香，制成葱油
* 加入豆腐，煸出水分，煸黄
* 加入酱油，炒香
* 加入水，盐
* 加入另一半葱
* 快出锅时加入淀粉收汁即可



**油->五花肉->葱->豆腐->酱油->水->盐->葱->淀粉**
