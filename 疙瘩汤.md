## 原料

| 原料   | 用量                         |
| ------ | ---------------------------- |
| 面粉   | 一勺半面粉                   |
| 鸡蛋   | 三个，一个拌疙瘩，两个打蛋花 |
|        |                              |
| 西红柿 | 切丁                         |
| 葱花   | 切葱花，多一些               |
|        |                              |
| 酱油   |                              |
| 盐     |                              |
|        |                              |
| 青菜   |                              |
| 香菜   |                              |





## 制法

* 鸡蛋中加入少量水，搅匀后拌疙瘩，一次少量倒入，搅拌，直至疙瘩成型
* 煸葱花，煸葱花至金黄色
* 加入西红柿、酱油，出炝锅味
* 加入水(热水、凉水均可)，烧开之后加入疙瘩，大块疙瘩用勺子磕成小块
* 加盐调味
* 加入青菜
* 打蛋花，鸡蛋加水搅拌均匀之后，转圈倒入锅中，稍成型之后搅拌
* 加入香菜出锅
